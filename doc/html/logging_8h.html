Function Documentation

void createLog ( char *  nFic)

File initialization.

The function creates the logging file and writes its header. If nFic is a null pointer or a null string, the file is created under a predefined name log.

The header consists of

  • a line title
  • a blank line
  • a quad line describing the meaning of the different fields of the state lines.

unsigned int  k,
FULL_STAT *  p_fSt

Writing the present full state as a double line at the end of the file.

If nFic is a null pointer or a null string, the lines are appended to a file under the predefined name log.

The following layout is obeyed for the full state in a double line

  • plane, porter, driver, queue and bus state
  • passengers state (p = 0,..., N-1).
