Detailed Description

Semaphore management.

(interface file)

Operations defined on semaphores:

  • creation of a set of semaphores
  • connection to a previously created set of semaphores
  • destruction of a previously created set of semaphores
  • signaling start of operations
  • down of a semaphore within the set
  • up of a semaphore within the set.
